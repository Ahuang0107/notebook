# 关于script标签和style标签加载顺序

## `<script>`元素

- `<script>`元素是网景公司创造的
- 因为页面要到在浏览器解析到<body>时才开始渲染，如果`<script>`放在`<head>`内，则先下载、解析和解释脚本，再渲染页面。——>就导致会在加载`<script>`
  时浏览器窗口空白。所以现代Web应用程序通常将`<script>`放在`<body>`中页面内容的后面，先渲染页面再加载脚本。
- defer属性：脚本会被延迟到整个页面都解析完毕后再运行。
